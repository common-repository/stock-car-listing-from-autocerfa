== Description ==
Autocerfa is the preferred software for car dealers in France. "Autocerfa Connector" plugin helps to display all vehicles on your WordPress website from Autocerfa. The usage of this plugin requires an account with Autocerfa platform and a list of vehicles in your stock.

"Autocerfa Connector" is maintained by OP Code Space LLC. Autocerfa has no responsibility of this plugin. We have permission to use their API to pull stock cars and displaying on user website.


== To activate this plugin you require: ==
1. An Autocerfa.com account version ELITE or ACCELERATEUR

2. To authorise this plugin to access Autocerfa with your username and password

If you require assistance, please email *support@opcodespace.com*

You will get all pro featured in free version except limited cars (max 10 cars). You will find more details on OP Code Space.

== Frequently Asked Questions ==

= How does this plugin work with Autocerfa? =

This plugin requires your authorization to synchronize all stock cars. Autocerfa provides only details of stock car and images. This plugin saves this information and images in your server.

= Is it secure? =

This plugin does not save your login credentials but Autocerfa token. Autocerfa provides only details of stock car and images. We recommend installing ssl in your server if not have.
